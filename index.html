About osu!droid

osu!droid is an unofficial osu project for android. This game was created many years ago by the osu! It is a free-to-play rhythm game for Android devices that involves clicking circles. community. Currently, a small group of fans and several foreign friends are working on this project together. Similar to it's original PC version, osu!droid's gameplay involves players tapping, sliding, and spinning on circles, sliders, and spinners to the beat of music. Mobile gamers that like difficult rhythm games are fond of this game. The osu! community has totally created a large number of songs and maps, which can be imported to osu!droid, and this game only has one game mode, including the osu!standard game mode.
